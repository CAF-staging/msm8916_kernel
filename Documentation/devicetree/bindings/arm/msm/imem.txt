Qualcomm IMEM

IMEM is fast on-chip memory used for various debug features and dma transactions.

Required properties

-compatible: "qti,msm-imem"
-reg: start address and size of imem memory

Example:

	qti,msm-imem {
		compatible = "qti,msm-imem";
		reg = <0xdeadbeef 0x1000>; /* < start_address size > */
	};
